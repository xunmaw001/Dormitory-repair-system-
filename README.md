# Dormitory-repair-system-
为优化宿舍生活体验，我们推出基于微信小程序的宿舍报修系统。它操作便捷，学生无需下载其他应用，在微信内即可完成报修。进入小程序，可快速填写宿舍号、故障类型、故障详情，还能上传图片辅助说明。提交后，维修人员将及时收到信息并安排维修。学生可随时查看报修进度，维修完成后还能评价反馈。该系统打破时空限制，大大提升报修效率，让宿舍设施问题得到及时解决 。
